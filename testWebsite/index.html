Click map to set: Start

How to use

  1. Click on the map to set a Start point (snapped to nearest path node).
  2. Click again to set an End point.
  3. The shortest path along mapped paths will be drawn with total distance.
  4. Use the Set End / Set Start button to switch modes, and Clear to reset.
About the data

This tool builds a routing graph from LineString path data in campus.geojson (e.g., highway=footway/path/steps, etc.). If some intended paths are missing from the GeoJSON, routing may not connect through those areas.
